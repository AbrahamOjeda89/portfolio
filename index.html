Imagina 3D

Proyecto creado para una práctica del ciclo formativo de DAW, que consiste en la creación completa de una página web mediante HTML5, CSS3 y JavaScript. En este projecto se plantea una página web de afiliados de Amazon en la que se habla sobre el mundo de la impresión 3D. Se dan recomendaciones sobre qué tipo y qué modelo de impresora elegir con enlaces directos a Amazon. Al entrar en Amazon a través de estos enlaces todo lo que se compre durante las 24 horas siguientes, esta web se queda un porcentaje de la venta.

Proyecto en Bootstrap

Creación de un proyecto con HTML, CSS y Bootstrap, en el que se utilizan muchas de sus clases. En esta web se puede ver una tabla creada con DataTable en la que se utilizan varias funcionalidades como un buscador, modificar la cantidad de registros a mostrar, pasar a la página deseada, y un botón de exportar que accede a un archivo modal. En este PopUp se puede elegir la cantidad de datos y el formato a descargar. También se utilizan diferentes tarjetas con enlaces externos. Por último, podemos ver otra tabla que se alimenta de una Api del tiempo en el que se muestra el tiempo de unas ciudades por defecto, pero en la parte de abajo se puede elegir la ciudad deseada.
